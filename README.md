# Campus Wellness Platform 

An anonymous campus mental wellness web application that allows students to book counseling sessions and communicate securely with counselors in real time.

---

## Features

- Separate login/signup for students and counselors  
- Anonymous counseling session booking  
- Real-time chat using Socket.IO  
- Secure and simple user interface  
- Session-based communication  

---

## Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- Axios
- Socket.IO Client

**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.IO
